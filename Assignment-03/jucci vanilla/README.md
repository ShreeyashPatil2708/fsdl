# MARKET

A lightweight fashion e-commerce website created using **HTML, CSS, and Vanilla JavaScript**. The project focuses on building an interactive shopping experience using only core web technologies, without relying on frameworks or external libraries.

## Highlights

- Browse products across multiple categories
- Filter items by Women, Men, and Accessories
- Interactive shopping cart with live updates
- Slide-out cart interface
- Step-by-step checkout process
- Multiple payment methods:
  - Credit/Debit Card
  - UPI
  - Cash on Delivery
- Automatic formatting for card details
- Smooth scroll-based reveal animations
- Responsive layout for different screen sizes

## Built With

- HTML5
- CSS3
- JavaScript (ES6)

## Running the Project

No installation or setup is required.

Simply download or clone the repository and open **`index.html`** in your preferred web browser.

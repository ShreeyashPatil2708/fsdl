# Assignment 03 — MARKET: Fashion E-Commerce SPA

A luxury fashion store built as a single-page app using only HTML, CSS, and vanilla JS — no frameworks.

## Features
- Client-side routing across Home, Collection, Campaign, and Checkout pages
- Category filter (All / Women / Men / Accessories)
- Slide-out cart drawer with live quantity controls
- 3-step checkout (Delivery → Payment → Review) with card number formatting
- Scroll-reveal animations via IntersectionObserver

## Stack
HTML · CSS · Vanilla JS

## Run
Open `jucci vanilla/index.html` directly in a browser — no build step needed.

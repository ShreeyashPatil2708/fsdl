# Assignment 06 — Appointment Booking System

Full-stack MERN app for booking medical appointments. Patients browse doctors, pick time slots, and manage bookings. Admins get a management panel.

## Features
- JWT auth with role-based access (patient / admin)
- Browse doctors and available time slots
- Book, reschedule, and cancel appointments
- Admin panel to manage doctors and all appointments

## Stack
React 18 (Vite) · Tailwind CSS · Node.js · Express · MongoDB · JWT

## Run
```bash
# Backend
cd appointment-booking-system/backend && npm install && npm run dev

# Frontend
cd appointment-booking-system/frontend && npm install && npm run dev
```

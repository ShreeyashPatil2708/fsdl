# Assignment 02 — Personal Portfolio

A personal developer portfolio built with plain HTML, CSS, and vanilla JS.

## Features
- Sticky navbar with scroll-aware active link highlighting
- Hero, About, Education, Skills, Projects, and Contact sections
- Mobile hamburger menu
- Smooth scroll with IntersectionObserver

## Stack
HTML · CSS · Vanilla JS

## Run
Open `portfolio/index.html` directly in a browser — no build step needed.
